# Ajudante de Orações

Criado por Rafael Alves Faria.

O programa foi criado por mim no 1º período da minha faculdade de Engenharia de Computação,
com o intuito de facilitar o acesso a diversas orações 
para auxiliar no momento das orações cotidianas.

[![Meu Vídeo](https://img.youtube.com/vi/CKsEtkZmz2E/mqdefault.jpg)](https://www.youtube.com/watch?v=CKsEtkZmz2E)

## Como usar?

Ao executar o programa em algum compilador de C, irá aparecer um menu no terminal. Para prosseguir deve escolher uma das opções apresentadas:
- 1 (Orações simples);
- 2 (Terço);
- 0 (Sair);

### Orações Simples:
Caso o usuário escolha a primeira opção, um novo menu irá aparecer no terminal, permitindo que o usuário selecione uma entre as orações cadastradas. Sendo elas:

- Pai Nosso;
- Ave Maria;
- Santo Anjo;
- Oração ao Espírito Santo;
- Credo Niceno Constantinopolitano;

Ao selecionar uma dessas orações, irá aparecer a mesma escrita, em português brasileiro, no terminal e no final a fonte de onde foi tirada o texto da oração.
Para retornar ao menu anterior deve-se digitar 0, como explicitado no menu apresentado no terminal.

### Terço:
Caso o usuário escolha a segunda opção, ele poderá optar em rezar o Santo Rosário ou o Santo Terço.
Caso escolha o Santo Terço, será perguntado qual o mistério do dia. 

Dessa forma, tanto para o Rosário quanto para o Terço (tendo já escolhido o mistério), será mostrado no terminal as orações que se devem ser rezadas para continuar rezando. Ao terminar a oração, o usuário deve apertar a tecla "Enter" e então a oração irá prosseguir até o fim do Rosário ou do Terço.

Vale ressaltar que, as Ave Maria aparecem numeradas para facilitar ao usuário.

### Sair:
Caso o usuário decida sair, uma mensagem de despedida irá aparecer no terminal e o programa irá finalizar.

# Limitações:

- Orações somente em português;
- Funciona somente no terminal;
